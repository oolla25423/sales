# Система управления данными о продажах

Django-приложение для управления данными о продажах с функциями импорта/экспорта в форматах XML и JSON. Приложение работает полностью через файловую систему без использования базы данных.

## Особенности

- Добавление данных о продажах через HTML-форму
- Сохранение данных в формате JSON или XML
- Загрузка существующих файлов JSON/XML
- Просмотр всех данных о продажах из сохраненных файлов
- Скачивание и удаление файлов
- Удаление отдельных записей о продажах из файлов

## Требования

- Python 3.8+
- Django 5.2+

## Установка

1. Установите необходимые пакеты:
   ```
   pip install -r requirements.txt
   ```

2. Создайте директорию для загрузок (если не существует):
   ```
   mkdir uploads
   ```

3. Запустите сервер разработки:
   ```
   python manage.py runserver
   ```

## Использование

1. Откройте приложение по адресу http://127.0.0.1:8000/
2. Добавьте новые данные о продажах с помощью формы
3. Выберите сохранение в формате JSON или XML
4. Загрузите существующие файлы JSON/XML
5. Просматривайте, скачивайте или удаляйте существующие файлы

## Структура данных

Данные о продажах включают:
- Название продукта
- Количество
- Цена (в рублях)
- Дата продажи
- Имя клиента
- Электронная почта клиента

## Хранение файлов

Все данные хранятся в каталоге `uploads` в виде JSON и XML файлов. Приложение не использует базу данных.
