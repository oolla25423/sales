# Система управления данными о продажах

Это простое веб-приложение на Django для управления данными о продажах. Создано как учебный проект для университета.

## Что делает приложение

- Добавляет информацию о продажах через веб-форму
- Сохраняет данные в базе данных
- Позволяет просматривать, редактировать и удалять записи
- Имеет поиск по записям

## Установка и запуск

1. Установите Python 3.8 или выше
2. Создайте виртуальное окружение:
   ```
   python -m venv venv
   ```
3. Активируйте виртуальное окружение:
   - Windows:
     ```
     venv\Scripts\activate
     ```
   - macOS/Linux:
     ```
     source venv/bin/activate
     ```
4. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
5. Создайте базу данных:
   ```
   python manage.py migrate
   ```
6. Запустите сервер:
   ```
   python manage.py runserver
   ```
7. Откройте в браузере: http://127.0.0.1:8000/

## Использование

- На главной странице вы можете добавить новую запись о продаже
- Просмотрите все записи в таблице
- Используйте поиск для быстрого нахождения записей
- Нажмите "Редактировать" для изменения записи
- Нажмите "Удалить" для удаления записи

## Структура записи о продаже

Каждая запись содержит:
- Название продукта
- Количество
- Цена
- Дата продажи
- Имя клиента
- Email клиента

## Файлы проекта

- `sales_data/` - основное приложение Django
- `sales_project/` - настройки проекта
- `manage.py` - утилита управления Django
- `requirements.txt` - список зависимостей
